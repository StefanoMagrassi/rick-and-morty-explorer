# Rick & Morty Explorer

Explore "Rick & Morty" tv-show characters.

### Install dependencies

```sh
$ npm ci
```

### Run tests

```sh
$ npm test
```

### Start dev server

```sh
$ npm start
```

and then visit [http://localhost:1234](http://localhost:1234/)
